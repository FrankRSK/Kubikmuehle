# 3D Vier Gewinnt - Kubikmühle

Browserbasiertes 3D-Spiel, inspiriert von "Vier gewinnt", auf einem 4×4×4-Spielfeld.

## Spielziel
Bringe vier Kugeln in einer Reihe (horizontal, vertikal oder diagonal) untergebracht.

## Features
- 3D-Darstellung mit Three.js
- KI-Gegner mit drei Schwierigkeitsstufen (Einfach, Mittel, Schwer/Strategisch)
- Soundeffekte via Web Audio API
- Desktop- und Mobile-Steuerung (Maus, Touch, Pinch-Zoom)
- Mehrsprachige Benutzeroberfläche (DE, EN, ES, CN)

## Schnellstart
Öffne `index.html` im Browser.

## Bedienung
- **Desktop:** Klick auf Stange zum Platzieren, Maus ziehen zum Drehen, Scrollen zum Zoomen
- **Mobile:** Wischen zum Drehen, Pinch zum Zoomen, Tippen auf Stange zum Platzieren

## Technik
- Einzelne `index.html` (CSS + JS eingebettet)
- Three.js r128 via CDN
- Keine Build-Tools erforderlich
