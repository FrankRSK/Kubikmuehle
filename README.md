# Kubikmühle 🧊 – 3D Vier Gewinnt

> Browserbasiertes 3D-Spiel, inspiriert von **"Vier gewinnt"**, auf einem 4×4×4-Spielfeld.

## Spielziel
Bringe vier Kugeln in eine Reihe – horizontal, vertikal oder diagonal in **drei Dimensionen**.

## Features
- 🎮 **3D-Spielfeld** mit Three.js (dreh- und zoombar)
- 🤖 **KI-Gegner** in drei Schwierigkeitsstufen (Einfach, Mittel, Strategisch)
- 🔊 **Soundeffekte** via Web Audio API
- 📱 **Desktop & Mobile** (Maus, Touch, Pinch-Zoom)
- 🌍 **Mehrsprachig** – Deutsch, Englisch, Spanisch, Chinesisch

## Schnellstart

```bash
git clone https://github.com/FrankRSK/Kubikmuehle.git
cd Kubikmuehle
# index.html im Browser öffnen – fertig!
```

## Technik

| Komponente | Technologie |
